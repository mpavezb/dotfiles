## Keybindings

### Tabs

| Key          | Description           |
| ------------ | --------------------- |
| Shift+Down   | New tab               |
| Shift+Left   | Go to left tab        |
| Shift+Right  | Go to right tab       |
| Ctrl+Left    | Move tab to the left  |
| Ctrl+Right   | Move tab to the right |
| Ctrl+d       | Close tab             |

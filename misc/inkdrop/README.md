# Inkdrop

**Keybindings**:
- Copy keybindings configuration file into `~/.config/inkdrop/`.

**Plugins**:
- math
- mermaid
- spell-checker

**Settings**:
- Themes:
  - Default Dark UI
  - Default Dark Syntax
- Editing:
  - Toolbar: Off
  - Line Numbers: Off
  - Highlight Active Line: On
  - Unordered List Bullet: "-
